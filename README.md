🎨 AI Image Generator (Flask Project)

This is a simple and interactive AI Image Generator Web App built using Flask.
Users can enter a text prompt and generate AI-based images instantly.

---

🚀 Features

- 🧠 Generate images using AI API
- ✍️ User-friendly input form
- 🖼️ Display generated image instantly
- ⬇️ Download generated image
- 🌐 Simple and clean UI

---

🛠️ Tech Stack

- Backend: Python (Flask)
- Frontend: HTML, CSS
- API: RapidAPI (AI Image Generator)

---

⚙️ How to Run the Project

1. Clone the repository:

git clone https://github.com/your-username/ai-image-generator-project.git

2. Go to project folder:

cd ai-image-generator-project

3. Install dependencies:

pip install flask requests

4. Run the app:

python app.py

5. Open browser:

http://127.0.0.1:5001/

---

🔑 Important Note

- Replace your RapidAPI key in "app.py":

"x-rapidapi-key": "YOUR_API_KEY"

⚠️ Never share your API key publicly.

---

📸 How it Works

1. User enters a prompt (e.g., "A cat flying in space")
2. Flask sends request to AI API
3. API returns generated image
4. Image is displayed on screen
5. User can download the image
